
# BE-FabelApps

A brief description of what this project does and who it's for US




## Run Locally

Clone the project

```bash
  git clone https://github.com/Rizki666/BE-FabelApps.git
```

Go to the project directory

```bash
  cd BE-FabelApps
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
